# Car Rental System - Java

Projeto desenvolvido em Java utilizando Programação Orientada a Objetos.

## Conceitos aplicados

- Classes e Objetos
- Encapsulamento
- Composição
- Serviços
- Regra de negócio
- Datas com LocalDateTime

## Funcionalidades

- Cálculo de aluguel por hora
- Cálculo de aluguel diário
- Aplicação de imposto
- Geração de nota de pagamento

## Tecnologias

- Java 17
- IntelliJ IDEA
